## How to use backend?

### Grab all package dependencies

```sh
go mod tidy
```

### Build program

```sh
go build .
```

### Run program

**NOTE**: Depend on your operating system it will generate different binary files.

+ If Unix-like

```sh
./commercial-shop.com
```

+ If Windows

```sh
./commercial-shop.com.exe
```

### Clean binary file

```sh
go clean
```
