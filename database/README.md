# Database

Written for Postgresql database

## Coding guidelines

### SQL syntax

Have to be all uppercase

```
CREATE FUNCTION
```

### Table

Have to be title each words

```
Category
ProductDetail
```

### Attribute and column

Have to be all lowercase

```
category_id
data_id
```

### Function

Title at first character but have `_` to seperate each words

```
Insert_category_all
```
